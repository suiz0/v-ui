# v-ui
Simple bootstrap vue ui components

# Usage

Add the reference to the library at the body's bottom


    <!DOCTYPE html>
    <html>
        ...
        <body>
        ...
        <script src="v-ui.js">
        </body>
    </html>


# Dependencies
* vuejs >= 2
* bootstrap-vue >= 2
